# stewpot/feeds

This package provides utilities for consuming feeds of different kinds
(RSS/Atom/JSON). It can also run as a standalone CLI program, if you have
[deno installed](https://docs.deno.com/runtime/getting_started/installation/) on
your system.

## CLI Install

To create a fresh installation of the CLI program, run the command displayed
below.

```sh
deno install -RWNE -g jsr:@stewpot/feeds/cli
```

## CLI Update

Update to newer versions by running the following in your terminal.

```sh
deno install --reload -f -RWNE -g jsr:@stewpot/feeds/cli
```
